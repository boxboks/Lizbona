Potrzebne pluginy w VSCode:
- Live Sass Compiler

[node.js](https://nodejs.org/en) - (wymagane)

Aby wszystko działało na stronie należy:
- zaimportować plik "cenzor.sql" do bazy odanych
- Press "Watch Sass"
- wpisać w terminal:
- `npm install`
- `node index.js`;
- strona powinna być na `localhost:3000`
